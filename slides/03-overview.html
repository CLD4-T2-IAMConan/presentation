PROJECT OVERVIEW

Passit이란?

정가 기반, 안전한 1:1 중개를
실시간으로 제공하는 티켓 거래 플랫폼

프리미엄 티켓 거래의 불투명성을 해소하고, 사용자 간 신뢰를 보장하는 중개 서비스입니다.

MSA 기반 독립 배포

마이크로서비스 아키텍처를 도입하여 서비스별 독립적인 배포와 유연한 스케일링이 가능합니다.

이벤트 기반 비동기 통신

서비스 간 느슨한 결합을 위해 SNS/SQS를 활용한 이벤트 기반 비동기 통신 체계를 구축했습니다.

실시간 인터랙션

WebSocket을 통한 실시간 채팅과 거래 상태 변경에 따른 즉각적인 시스템 알림을 지원합니다.
