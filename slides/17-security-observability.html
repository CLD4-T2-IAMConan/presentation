SECURITY & OBSERVABILITY

보안 및 관찰성

네트워크 보안

  • Security Groups: 포트 및 프로토콜 제어
  • Private Subnets: 데이터베이스 및 내부 서버 격리
  • NACL: 서브넷 단위 트래픽 필터링
  • VPC Endpoints: AWS 서비스 프라이빗 접근

데이터 보안

  • KMS: 데이터 암호화 키 중앙 관리
  • Encryption: RDS(저장 데이터) 및 S3(객체) 암호화
  • Secrets Manager: DB 자격 증명 등 민감 정보 관리
  • TLS 1.3: 전송 데이터 암호화

접근 제어

  • IAM & IRSA: Pod 단위의 정교한 권한 관리
  • 최소 권한 원칙: 서비스별 필요 권한만 부여
  • GitHub OIDC: CI/CD 파이프라인 인증 강화
  • RBAC: Kubernetes 리소스 접근 제어

관찰성 (Observability)

  • Prometheus: 애플리케이션 및 인프라 메트릭 수집
  • CloudWatch Logs: 로그 중앙화 및 보관
  • Grafana: 실시간 대시보드 시각화
  • AlertManager: 장애 발생 시 실시간 알림 발송
