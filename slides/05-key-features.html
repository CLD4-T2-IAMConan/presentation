KEY FEATURES

주요 특징 하이라이트

마이크로서비스 아키텍처

서비스별 독립적인 배포와 확장이 가능한 MSA 구조를 채택하여 유연성을 확보했습니다.

이벤트 기반 비동기 통신

SNS/SQS를 활용하여 서비스 간 결합도를 낮추고 비동기 처리를 통해 성능을 최적화했습니다.

실시간 채팅 기능

WebSocket과 STOMP 프로토콜을 사용하여 구매자와 판매자 간의 실시간 소통을 지원합니다.

고가용성 인프라

AWS Multi-AZ 배포 및 자동 스케일링을 통해 장애 상황에서도 안정적인 서비스를 보장합니다.

자동화된 CI/CD

GitHub Actions와 ArgoCD를 연동하여 GitOps 기반의 자동화된 배포 파이프라인을 구축했습니다.

모니터링 및 알림

Prometheus와 Grafana를 통한 실시간 모니터링 및 이상 징후 발생 시 즉각적인 알림을 제공합니다.
